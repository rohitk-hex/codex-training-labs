# Documentary Prompts

These prompts guide technical storytellers in framing how the React front-end and Express backend collaborate.

1. **Architecture narrative**: Sketch a documentary arc that starts with the business need (Myntra-style storefront), explains the challenge of curating festival-ready products, and resolves with the React/UI + Express catalog pairing.
2. **Technical personas**: Document the designer (CSS/hero layout), the front-end engineer (React components, hooks), and the backend engineer (Express router, price formatter). Cite the relevant files or functions they would host.

4. **Observability cutaways**: What telemetry, logging, or HTTP status messaging would you highlight to show the production readiness of this repo? Mention the hero status card, fetch error handling, or console logging.
5. **Next experiments**: What future automations, testing harnesses, or integrations would you invite viewers to explore after watching the documentary?
