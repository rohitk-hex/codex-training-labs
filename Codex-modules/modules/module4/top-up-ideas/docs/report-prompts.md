# Report Writing Prompts

These prompts structure a short-form technical report after you explore or extend the Myntra-style stack.

1. What did you inspect during this pass (files, services, endpoints, tests)? Mention the frontend entry points, the Express API, and any docs you referenced.
2. What technical challenges or surprises surfaced (CORS, fetch failures, layout regressions)? Include console logs, network errors, or build warnings that you encountered.
3. What design or implementation decisions did you make? Tie them back to the Indian context, the React layout, or the catalog API responses.
4. How did you validate the changes (manual browser testing, Lighthouse, API checks)? Supply commands or tools used and the outcomes observed.
5. What unanswered questions or next steps remain for future contributors? Link these to the prompt docs in this folder if relevant.
