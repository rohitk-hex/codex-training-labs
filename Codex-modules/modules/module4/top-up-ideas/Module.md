# Module 4 — React + Express Codebase Understanding

This module guides learners through the Myntra-inspired React/Vite frontend plus the supporting Node/Express catalog API, along with the documentation prompts.

## Pass Structure
1. **First pass**: Launch the backend (ackend/index.js) and the frontend (rontend/src/App.jsx). Inspect how the Express route shapes the response and how React consumes it.
2. **Second pass**: Use the prompts under docs/ to document the component hierarchy, data flow, and the technical story for future engineers. Each prompt scales to larger repositories by focusing on entry points and runtime scripts.
3. **Third pass**: Apply an idea from modification-ideas.md to extend the experience, then record the change in the reporting prompts so the cohort can share the technical insights.

## Artifacts
- myntra-style-store/frontend: React code and styles built with Vite.
- myntra-style-store/backend: Express catalog API.
- docs/: Prompt suites for codebase summaries, documentaries, and reports.
- modification-ideas.md: Ideas for backend or frontend enhancements.
